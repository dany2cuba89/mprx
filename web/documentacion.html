Manual de Usuario

Bienvenido a la documentación de EmpreX. Aquí encontrarás una guía detallada para utilizar todas las funcionalidades de nuestra plataforma. Sigue los pasos a continuación para sacar el máximo provecho de EmpreX.

▶ 1. Inicio de Sesión

Para acceder a la plataforma EmpreX, primero debes iniciar sesión. Una vez que se haya realizado la contratación del servicio, se te proporcionará un correo electrónico y una contraseña para acceder.

1. Acceso a la Página de Inicio de Sesión

Para acceder a la página de inicio de sesión, abre tu navegador web e ingresa la URL de la plataforma EmpreX. Si ya estás en la página principal, haz clic en el botón "Iniciar Sesión" o en el enlace correspondiente.

2. Campos del Formulario

La página de inicio de sesión tiene dos campos principales que debes completar:

  • Correo Electrónico: Ingresa el correo electrónico que te proporcionó EmpreX.
  • Contraseña: Ingresa la contraseña que te proporcionó EmpreX.

3. Iniciar Sesión

Una vez que hayas completado ambos campos, haz clic en el botón "Entrar". Si tus credenciales son correctas, serás redirigido automáticamente al Dashboard de EmpreX.

4. Manejo de Errores

Si ocurre un error al intentar iniciar sesión, verás un mensaje de alerta en la parte superior del formulario. Los errores más comunes son:

  • Credenciales Incorrectas: Asegúrate de que el correo electrónico y la contraseña estén escritos correctamente.
  • Cuenta No Registrada: Si no tienes una cuenta en EmpreX, contacta al administrador de la plataforma.

▶ 2. Guía de Uso del AppBar (Barra de Navegación)

El AppBar es la barra de navegación superior de la aplicación. Aquí te explicamos cómo usarlo:

1. Título de la Aplicación

El título "EmpreX" aparece en la esquina superior izquierda. Haz clic en él para volver a la página de inicio.

2. Avatar del Usuario

En la esquina superior derecha, verás un círculo con la primera letra de tu nombre de usuario. Haz clic en él para abrir el menú de usuario.

3. Menú de Usuario

Al hacer clic en el avatar, se abrirá un menú desplegable con las siguientes opciones:

  • Información del Usuario: Verás tu correo electrónico y un saludo personalizado.
  • Editar Perfil: Haz clic en el ícono de lápiz para editar tu perfil.
  • Cerrar Sesión: Cierra tu sesión actual y te redirige a la página de inicio de sesión.
  • Enlaces de Navegación: Accesos directos a las diferentes secciones de la aplicación (Inicio, Caja, Inventario, etc.).

4. Navegación Según Tu Rol

Dependiendo de tu rol (dueño, administrador, económico o cajero), verás diferentes opciones en el menú. Si no ves una sección, es porque no tienes permiso para acceder a ella.

5. Cómo Cerrar el Menú

Puedes cerrar el menú de usuario haciendo clic fuera de él o seleccionando una opción.

▶ 3. Gestión de Perfil de Usuario

La página de perfil de usuario te permite actualizar tu información personal, como tu nombre, correo electrónico, nombre de usuario y contraseña. Aquí te explicamos cómo utilizarla:

1. Acceso a la Página de Perfil

Para acceder a la página de perfil, haz clic en tu avatar en la esquina superior derecha de la aplicación y selecciona "Editar Perfil" en el menú desplegable.

2. Campos del Formulario

En la página de perfil, encontrarás los siguientes campos:

  • Nombre Completo: Ingresa tu nombre completo.
  • Nombre de Usuario: Ingresa tu nombre de usuario.
  • Correo Electrónico: Ingresa tu correo electrónico.
  • Contraseña: Ingresa una nueva contraseña si deseas cambiarla (opcional).

3. Actualizar Perfil

Para guardar los cambios, haz clic en el botón "Actualizar Perfil". Si los datos se actualizan correctamente, verás una notificación de éxito.

4. Notificaciones

Después de realizar cambios, se mostrará una notificación indicando si la actualización fue exitosa o si ocurrió un error.

5. Manejo de Errores

Si ocurre un error al actualizar tu perfil, verás un mensaje de error en la notificación. Algunos errores comunes incluyen:

  • Correo Electrónico Ya Registrado: El correo electrónico que ingresaste ya está en uso.
  • Nombre de Usuario Ya Existente: El nombre de usuario que ingresaste ya está en uso.
  • Contraseña Inválida: La contraseña no cumple con los requisitos de seguridad.

▶ 9. Gestión de Inventario

La página de gestión de inventario te permite administrar los productos registrados en la plataforma. Aquí te explicamos cómo utilizarla:

1. Acceso a la Página de Gestión de Inventario

Para acceder a la página de gestión de inventario, selecciona la opción "Inventario" en el menú de navegación.

2. Lista de Productos

En la página de gestión de inventario, verás una tabla con la lista de productos registrados. Cada fila muestra:

  • Nombre: El nombre del producto.
  • Unidad de Medida: La unidad de medida del producto (kg, litros, unidades, etc.).
  • Precio de Compra: El precio al que se compró el producto.
  • Precio de Venta: El precio al que se vende el producto.
  • Stock: La cantidad disponible del producto.
  • Nivel Mínimo: El nivel mínimo de stock permitido para el producto.
  • Categoría: La categoría a la que pertenece el producto.

3. Acciones Disponibles

Desde esta página, puedes realizar las siguientes acciones:

  • Exportar a Excel: Haz clic en el botón "Exportar a Excel" para descargar un archivo Excel con el inventario actual.
  • Descargar Código QR: Selecciona un producto y haz clic en el botón "Descargar Código QR" para generar y descargar un código QR asociado al producto. Este código QR se utiliza para identificar rápidamente el producto en el módulo de APV (Caja) al escanearlo.

4. Exportar a Excel

Para exportar el inventario a un archivo Excel, sigue estos pasos:

  1. Haz clic en el botón "Exportar a Excel".
  2. Se descargará automáticamente un archivo Excel con la lista de productos, incluyendo detalles como nombre, precio de compra, precio de venta, stock, nivel mínimo, unidad de medida y categoría.

5. Descargar Código QR

Para generar y descargar un código QR asociado a un producto, sigue estos pasos:

  1. Selecciona un producto de la lista.
  2. Haz clic en el botón "Descargar Código QR".
  3. Se generará y descargará automáticamente un archivo PNG con el código QR del producto.

Nota: El código QR de cada producto contiene su identificador único. Este código se utiliza en el módulo de APV (Caja) para escanear y agregar rápidamente el producto a una venta, agilizando el proceso de facturación.

6. Notificaciones

Después de realizar cualquier acción, se mostrará una notificación indicando si la operación fue exitosa o si ocurrió un error.

7. Manejo de Errores

Si ocurre un error durante la gestión de inventario, verás un mensaje de error en la notificación. Algunos errores comunes incluyen:

  • Error al Exportar: No se pudo generar el archivo Excel. Asegúrate de tener conexión a Internet y permisos para descargar archivos.
  • Error al Generar QR: No se pudo generar el código QR. Verifica que el producto seleccionado tenga un identificador válido.

▶ 10. Gestión de Caja Registradora

El módulo de Caja Registradora te permite gestionar las ventas de productos en tu negocio. Aquí te explicamos cómo utilizarlo:

1. Acceso a la Página de Caja Registradora

Para acceder a la página de caja registradora, selecciona la opción "Caja" en el menú de navegación.

2. Interfaz de Caja Registradora

En la página de caja registradora, encontrarás las siguientes secciones:

  • Carrito de Compras: Muestra los productos añadidos a la venta actual, con detalles como nombre, precio, cantidad e importe.
  • Total de la Venta: Muestra el monto total de la venta actual.
  • Método de Pago: Permite seleccionar el método de pago (efectivo o transferencia).
  • Botones de Acción: Incluye botones para agregar productos, confirmar la venta y reiniciar el proceso.

3. Agregar Productos al Carrito

Para agregar productos al carrito, sigue estos pasos:

  1. Haz clic en el botón "Agregar Producto".
  2. Se abrirá un modal con dos pestañas: Manual y QR.
  3. Pestaña Manual:
    • Busca productos por nombre utilizando el campo de búsqueda.
    • Selecciona un producto de la lista y especifica la cantidad.
    • Haz clic en "Añadir al Carrito" para agregar el producto.
  4. Pestaña QR:
    • Escanea el código QR del producto utilizando la cámara.
    • El producto se agregará automáticamente al carrito si está disponible en el inventario.

4. Modificar Cantidades en el Carrito

Para modificar las cantidades de los productos en el carrito:

  • Utiliza el campo de selección en la columna "Cantidad" para ajustar la cantidad de cada producto.
  • Si la cantidad se establece en 0, el producto se eliminará del carrito.

5. Seleccionar Método de Pago

Antes de confirmar la venta, selecciona el método de pago:

  • Efectivo: No se requieren detalles adicionales.
  • Transferencia: Debes ingresar el número de tarjeta, nombre del cliente y CI del cliente.

6. Confirmar Venta

Para finalizar la venta, sigue estos pasos:

  1. Verifica que todos los productos y cantidades en el carrito sean correctos.
  2. Selecciona el método de pago y completa los detalles si es necesario.
  3. Haz clic en el botón "Confirmar Venta".
  4. Si la venta se procesa correctamente, se mostrará una notificación de éxito y el carrito se vaciará automáticamente.

7. Notificaciones

Durante el proceso de venta, se mostrarán notificaciones para informarte sobre el estado de las operaciones, como:

  • Éxito: Cuando un producto se agrega correctamente al carrito o la venta se confirma.
  • Error: Si ocurre un problema, como un producto sin stock o un error en el proceso de pago.
  • Información: Para notificaciones generales, como la eliminación de un producto del carrito.

8. Manejo de Errores

Si ocurre un error durante el proceso de venta, verás un mensaje de error en la notificación. Algunos errores comunes incluyen:

  • Producto sin Stock: No se puede agregar un producto que no tiene stock disponible.
  • Error en el Pago: Si el método de pago seleccionado no es válido o falta información.
  • Error de Escaneo: Si el código QR escaneado no corresponde a un producto válido.

▶ 11. Gestión de Facturas

El módulo de Gestión de Facturas te permite consultar y gestionar las facturas generadas en tu negocio, ya sean de venta o de compra. Aquí te explicamos cómo utilizarlo:

1. Acceso a la Página de Gestión de Facturas

Para acceder a la página de gestión de facturas, selecciona la opción "Facturas" en el menú de navegación.

2. Interfaz de Gestión de Facturas

En la página de gestión de facturas, encontrarás las siguientes secciones:

  • Filtros: Permite seleccionar el tipo de factura (venta o compra) y el intervalo de fechas para filtrar los resultados.
  • Tabla de Facturas: Muestra las facturas generadas, con detalles como fecha, hora, tipo, total y cajero.
  • Paginación: Permite navegar entre las páginas de resultados.
  • Botones de Acción: Incluye botones para ver detalles de una factura y exportarla a Excel.

3. Filtrar Facturas

Para filtrar las facturas, sigue estos pasos:

  1. Selecciona el Tipo de Factura (venta o compra) en el menú desplegable.
  2. Selecciona el Intervalo de Fechas:
    • Hoy: Muestra las facturas generadas hoy.
    • Última Semana: Muestra las facturas de los últimos 7 días.
    • Último Mes: Muestra las facturas del mes anterior.
    • Último Año: Muestra las facturas del año anterior.
    • Personalizado: Permite seleccionar un rango de fechas específico.
  3. Si seleccionas Personalizado, ingresa las fechas de inicio y fin en los campos correspondientes.
  4. Las facturas se actualizarán automáticamente según los filtros seleccionados.

4. Ver Detalles de una Factura

Para ver los detalles de una factura, sigue estos pasos:

  1. Selecciona una factura de la tabla haciendo clic en ella.
  2. Haz clic en el botón "Detalles".
  3. Se abrirá un modal con la información detallada de la factura, incluyendo:
    • Fecha y Hora: Fecha y hora en que se generó la factura.
    • Total: Monto total de la factura.
    • Método de Pago: Método de pago utilizado (efectivo o transferencia).
    • Cajero: Nombre del cajero que generó la factura.
    • Productos: Lista de productos incluidos en la factura, con detalles como nombre, cantidad, precio e importe.

5. Exportar una Factura a Excel

Para exportar una factura a Excel, sigue estos pasos:

  1. Selecciona una factura de la tabla haciendo clic en ella.
  2. Haz clic en el botón "Exportar a Excel".
  3. Se descargará automáticamente un archivo Excel con los detalles de la factura seleccionada.

6. Notificaciones

Durante el proceso de gestión de facturas, se mostrarán notificaciones para informarte sobre el estado de las operaciones, como:

  • Éxito: Cuando una factura se exporta correctamente.
  • Error: Si ocurre un problema, como la falta de selección de una factura.

7. Manejo de Errores

Si ocurre un error durante la gestión de facturas, verás un mensaje de error en la notificación. Algunos errores comunes incluyen:

  • Factura No Seleccionada: Debes seleccionar una factura antes de realizar acciones como ver detalles o exportar.
  • Error de Exportación: Si no se puede generar el archivo Excel.

▶ 13. Gestión de Reportes

El módulo de Gestión de Reportes te permite generar y exportar diferentes tipos de reportes financieros y operativos para tu negocio. Aquí te explicamos cómo utilizarlo:

1. Acceso a la Página de Gestión de Reportes

Para acceder a la página de gestión de reportes, selecciona la opción "Reportes" en el menú de navegación.

2. Interfaz de Gestión de Reportes

En la página de gestión de reportes, encontrarás las siguientes secciones:

  • Pestañas de Reportes: Permite seleccionar el tipo de reporte que deseas generar (Balance General, Compras, Ventas, Pagos a Empleados, Productos Más Vendidos, Alertas de Stock Bajo).
  • Filtros: Permite seleccionar un intervalo de fechas (hoy, última semana, último mes, último año o personalizado) para filtrar los datos del reporte.
  • Tabla de Datos: Muestra los datos correspondientes al reporte seleccionado, con opciones de paginación para navegar entre los resultados.
  • Gráficos: En algunos reportes, como el Balance General, se muestran gráficos para visualizar los datos de manera más clara.
  • Botones de Acción: Incluye botones para exportar el reporte a Excel.

3. Seleccionar un Tipo de Reporte

Para seleccionar un tipo de reporte, sigue estos pasos:

  1. Haz clic en la pestaña correspondiente al reporte que deseas generar (por ejemplo, "Balance General").
  2. Selecciona el intervalo de fechas en el menú desplegable.
  3. Si seleccionas "Personalizado", ingresa las fechas de inicio y fin en los campos correspondientes.
  4. Los datos del reporte se actualizarán automáticamente según los filtros seleccionados.

4. Exportar un Reporte a Excel

Para exportar un reporte a Excel, sigue estos pasos:

  1. Selecciona el tipo de reporte que deseas exportar.
  2. Haz clic en el botón "Exportar a Excel".
  3. Se descargará automáticamente un archivo Excel con los datos del reporte seleccionado.

5. Tipos de Reportes Disponibles

Los tipos de reportes disponibles son:

  • Balance General: Muestra un resumen financiero con los ingresos, egresos, utilidad neta y utilidad neta después de pagos.
  • Compras: Muestra un resumen de las compras realizadas, incluyendo productos comprados, cantidades, precios unitarios y gastos totales.
  • Ventas: Muestra un resumen de las ventas realizadas, incluyendo productos vendidos, cantidades, precios unitarios e ingresos totales.
  • Pagos a Empleados: Muestra un resumen de los pagos realizados a los empleados, incluyendo montos, fechas y métodos de pago.
  • Productos Más Vendidos: Muestra un listado de los productos más vendidos, incluyendo cantidades vendidas e ingresos totales.
  • Alertas de Stock Bajo: Muestra un listado de productos con stock por debajo del nivel mínimo, incluyendo el stock actual y la diferencia con el nivel mínimo.

6. Notificaciones

Durante el proceso de generación de reportes, se mostrarán notificaciones para informarte sobre el estado de las operaciones, como:

  • Éxito: Cuando un reporte se exporta correctamente.
  • Error: Si ocurre un problema, como la falta de datos para el intervalo seleccionado.

7. Manejo de Errores

Si ocurre un error durante la generación de reportes, verás un mensaje de error en la notificación. Algunos errores comunes incluyen:

  • Datos No Disponibles: No hay datos para el intervalo de fechas seleccionado.
  • Error de Exportación: Si no se puede generar el archivo Excel.
